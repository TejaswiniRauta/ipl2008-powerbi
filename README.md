# 📊 IPL Data Analysis Dashboard (2008–2020)

## 🔍 Overview
This project is a Power BI dashboard built to analyze IPL matches data from 2008 to 2020.  
The goal is to extract meaningful insights about team performance, match locations, and winning patterns using interactive visualizations.

---

## 🎯 Problem Statement
Raw IPL data is difficult to interpret directly.  
This dashboard transforms unstructured match data into clear, interactive insights for better understanding and decision-making.

---

## 📊 Key Features
- 📅 Season-wise match analysis using dynamic slicers
- 🌍 City-wise distribution of matches
- 🏏 Team participation and performance insights
- 🏆 Winner-based filtering
- 📈 Interactive visuals for better exploration

---

## 🧠 Key Insights
- Certain cities hosted a significantly higher number of matches
- Some teams consistently performed better across seasons
- Match outcomes show patterns based on location and teams

---

## 🛠 Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Calculated Columns & Measures)
