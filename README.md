# IPL 2008 Power BI Dashboard

## 📊 Project Overview
This project analyzes the IPL 2008 dataset using Power BI to extract meaningful insights about team performance, player statistics, and match outcomes.

## 🛠 Tools Used
- Power BI
- Data Visualization

## 📈 Key Insights
- Top performing team of IPL 2008
- Highest run scorer in the tournament
- Toss winning impact on match results
- Win distribution across teams
